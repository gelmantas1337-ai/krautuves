TLDR:
Paprastą UI. (Start - Custom - Quit)
              Start - Programa parenka random žodį iš duomenų bazės. (reikės pasigilint bet čj read .txt file)
                      Custom - User pasirenkamas žodis.

UI elementai - Laikmatis - Pasikorėlis.
               Paprastas Laikmatis (Kiek užtruks user'io spėjimas)
                           Pasikorėlis - iš kelių dalių kuriamos krautuvės. Su kiekvienu blogu spėjimu piešiama viena dalis.
